# Soft Skills for the AI Era

An open-source trainer that helps professionals assess and develop soft skills critical in the age of AI. Take a self-assessment, get AI-powered analysis, test your knowledge with a quiz, and receive personalized homework assignments.

**Live demo:** [https://soft-skills.chillai.space](https://soft-skills.chillai.space)

## User Flow

1. **Context** (`/context`) -- Pick your profession, optionally upload a resume or paste a job description for personalized recommendations
2. **Topics** (`/topics`) -- Choose from 10 soft skill dimensions to focus on (AI can recommend topics based on your job description)
3. **Self-Assessment** (`/test`) -- Rate yourself on 5 situational questions per topic (1-5 scale, no right/wrong answers)
4. **Results** (`/results`) -- AI analysis with radar chart, strengths/weaknesses breakdown, and gap identification
5. **Quiz** (`/quiz`) -- Knowledge-based multiple-choice questions with explanations and source references
6. **Homework** (`/homework`) -- Personalized practice assignment generated by AI based on your weak areas, time budget, and context

## 10 Dimensions

| Dimension | Focus |
|---|---|
| Critical Thinking | Evaluating AI output, catching errors, not trusting blindly |
| Communication | Setting tasks for AI and people with equal clarity |
| Adaptability | Learning new tools faster than they become obsolete |
| Self-Organization | Staying disciplined when AI makes everything feel easy |
| Product Thinking | Focusing on user needs, not just code |
| Collaboration | Working in teams where AI does half the job |
| Leadership | Leading when AI cannot make the decision for you |
| Conflict Resolution | Arguing constructively -- with people and about AI decisions |
| Emotional Intelligence | Understanding people when work is increasingly with machines |
| Time Management | Not spending half a day prompting what you could do in an hour |

## Tech Stack

- **Next.js 14** (App Router) + TypeScript + Tailwind CSS
- **AI:** GPT-4o-mini (primary, cheapest) → Claude Haiku 4.5 (fallback)
- **Deployment:** Cloudflare Workers via `@opennextjs/cloudflare`
- **Storage:** Cloudflare KV (analytics), `sessionStorage` (all user data -- nothing stored server-side)
- **PDF parsing:** `unpdf` (server-side text extraction from resumes)

## Getting Started

### Prerequisites

- Node.js 18+
- At least one API key: [OpenAI](https://platform.openai.com/api-keys) or [Anthropic](https://console.anthropic.com/)

### Install and Run

```bash
git clone https://github.com/job-search-toolkit/soft-skills-check.git
cd soft-skills-check
npm install
cp .env.example .env.local
```

Edit `.env.local` — add at least one API key:

```bash
OPENAI_API_KEY=sk-proj-...        # GPT-4o-mini (primary, ~$0.15/1M tokens)
ANTHROPIC_API_KEY=sk-ant-...      # Claude Haiku (fallback, optional)
```

Start the dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Deploy to Your Own Cloudflare

```bash
# 1. Build for Cloudflare
npx opennextjs-cloudflare build --dangerouslyUseUnsupportedNextVersion

# 2. Deploy (first time creates the worker)
npx wrangler deploy

# 3. Set API key(s) as secrets
echo "your-key" | npx wrangler secret put OPENAI_API_KEY
echo "your-key" | npx wrangler secret put ANTHROPIC_API_KEY  # optional

# 4. Create KV namespace for analytics
npx wrangler kv namespace create ANALYTICS
# → Copy the ID into wrangler.jsonc under kv_namespaces

# 5. (Optional) Add custom domain
npx wrangler deploy --domain your-domain.example.com
```

### Testing Your Branch Before PR

```bash
# Verify build passes
npm run build

# Preview locally with Cloudflare runtime
npx opennextjs-cloudflare build --dangerouslyUseUnsupportedNextVersion
npx wrangler dev

# Or deploy a preview (*.workers.dev URL, doesn't touch production)
npx wrangler deploy
```

## Privacy

No user data is stored on the server. All assessment data lives in the browser's `sessionStorage` and is lost when the tab is closed. AI calls go to OpenAI or Anthropic for analysis/generation. Anonymous aggregate analytics (completion rates, question quality ratings) are stored in Cloudflare KV — no personal data.
